# Profile graphics setup

1. Copy every file in this package into the root of `UtkarshDubeyGIT/UtkarshDubeyGIT`, preserving the folders.
2. Open the repository's **Actions** tab and run **Generate profile graphics** once.
3. Wait for the workflow to create the `output` branch. The contribution card and Pac-Man animation will then appear in the README.
4. If the workflow cannot publish, open **Settings → Actions → General → Workflow permissions**, choose **Read and write permissions**, and run it again.

The workflow refreshes both graphics every day. It uses GitHub's own contribution API for the statistics card; no stats-widget host is required.
